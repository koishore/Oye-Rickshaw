Koishore's Ride Tracker



Endpoints:

/tracker/:

This endpoint allows you see all the drivers on the world map.
Click on any one of the cab icons to see the driver and ride details.

/nearby/:

This endpoint allows you see all drivers within 200m of a particular location.
It takes 2 arguments, latitude and longitude and returns all drivers within the 200m radius.
Currently, the database has 3 drivers only in Bangalore.
Sample inputs to see results:
latitude = 13.0551, longitude = 77.5995 will yield 2 results
latitude = 13.054578, longitude = 77.596316 will yield 2 results
